Excel - Dezembro 2024 à Janeiro 2025

Neste projeto, fui responsável por elaborar uma planilha no Excel que facilitou a mobilização de diversos treinamentos para colaboradores. Utilizando várias funções do Excel, como tabelas dinâmicas, fórmulas e formatação condicional, criei um plano de ação onde organizou e monitorou o progresso dos treinamentos. Essa planilha não apenas otimizou a gestão dos treinamentos, mas também garantiu que todos os colaboradores tivessem acesso às informações necessárias para seu desenvolvimento profissional.


Cidades Historicas

Guia Cidades Historicas- 2025

Este projeto foi desenvolvido utilizando as linguagens HTML e CSS com o objetivo de criar um guia interativo sobre três importantes cidades históricas de Minas Gerais: Diamantina, Ouro Preto e Tiradentes. O site apresenta informações detalhadas sobre a história, pontos turísticos, cultura e arquitetura dessas cidades, proporcionando aos usuários uma experiência rica e visualmente agradável. A estrutura do site é responsiva, garantindo uma navegação fácil e intuitiva em diferentes dispositivos. A escolha de HTML e CSS para este projeto reflete a intenção de criar um design limpo, organizado e de fácil acesso às informações, destacando a beleza e a importância histórica dessas cidades.
